But you could always look at Python, which is an entirely different game changer when it comes to Algo Trading.

What is Python?

Python is a free open-source and cross-platform language which has a rich library for almost every task imaginable and specialized research environment.

It is anexcellent choice for automated tradingin case of low/medium trading frequency, i.e. for trades which do not last less than a few seconds. It has multiple APIs/Libraries that can be linked to make it optimal, cheaper and allow greater exploratory development of multiple trade ideas.
