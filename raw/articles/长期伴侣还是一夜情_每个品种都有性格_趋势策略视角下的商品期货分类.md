计算的是“20日波动强度相对于20日净位移的比值”，代码如下：

```python
# 每日真实波幅，取日内振幅和相对昨收跳空幅度中的最大值
tr = max(high - low, abs(high - prev_close), abs(low - prev_close))
# 14日平均真实波幅
atr = rolling_mean(tr, 14)
# ATR占当前价格的比例
atr_pct = atr / close
# 20日价格净位移的绝对值
directional_move = abs(close.pct_change(20))
# 噪音比例=ATR相对波动 / 20日方向位移，并取全样本中位数
noise_ratio = median(atr_pct / directional_move)
```

![image](https://mmbiz.qpic.cn/mmbiz_png/S1iaf4GgGjEz18ZthVZQeeKiaBvyShcLuF2y0sLtzy87rm6qiaH8gtchSn17H16Fv1fgYKc9ibjK3kkT5UQQqQJyQemHNialqJ4l9FIGQHkzDzkg/640?wx_fmt=png&from=appmsg)

计算的是“为了走出一段趋势，过程中付出了多大的最大回撤代价”，代码如下：

```python
# 计算最大回撤
nav = close / close.iloc[0]
drawdown = nav / nav.cummax() - 1
max_drawdown = drawdown.min()
# 计算净趋势幅度
net_move = abs(close.iloc[-1] / close.iloc[0] - 1)
# 计算回撤结构比
abs(max_drawdown) / net_move
```

跳空次数计算“K线实体区间完全断开”的出现次数。

最后来算总账。我对跳空占比和回撤结构比赋予更高权重，各占40%，而将趋势流畅度和噪音比例各赋予10%。

原因在于趋势交易的核心不只是“能不能走出趋势”，更在于“这段趋势能不能被交易者真正拿住”。趋势策略的出场约束通常先来自两个方面：

止损位置是否容易被跳空穿透，跳空越多，止损执行的不确定性越强，真实亏损往往会偏离预期；

持仓过程中是否会因较大的中途回撤而被迫止损、减仓或提前离场。回撤结构越差，即使最终方向正确，过程中也更容易因为回撤过大而提前减仓、平仓
