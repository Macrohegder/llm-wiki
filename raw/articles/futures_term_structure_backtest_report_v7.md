# 期货期限结构价差反转策略 - 回测结果报告 v7

## 策略概述

**策略名称**: 期货期限结构价差反转策略 (Futures Term Structure Overreact)
**核心逻辑**: 当期限结构价差（F1-F2）偏离历史均值时，预期价差回归，做多低Z-Score品种、做空高Z-Score品种
**交易规则**: 等权重，双合约（F1+F2方向相反），持仓5个交易日
**品种池**: 51个活跃商品期货品种
**回测区间**: 2011-01-04 ~ 2026-04-30 (3721个交易日)
**初始资金**: 10,000,000元

## 关键发现

1. **策略有微弱正收益**: 年化0.43%-0.60%，扣除手续费后仍有正收益
2. **手续费侵蚀严重**: 万1手续费吃掉44.9%收益，万0.5吃掉22.45%
3. **风险较低**: 最大回撤仅2%左右，波动率约0.93%
4. **胜率接近随机**: 50-52%胜率 + 1.07盈亏比，优势微弱
5. **因子信号质量一般**: 51个品种中仅23个有正多空差，平均为负

## 与原文对比

原文（中国商品期货2010-2017，30品种）:
- 年化收益: ~8-12%
- 夏普比率: ~0.8-1.2

本回测（2011-2026，51品种）:
- 年化收益: 0.43%-0.60%
- 夏普比率: 0.46-0.64

**差异原因分析**:
1. 品种池扩大后，边缘品种拉低收益
2. 2020年后市场效率提升，期限结构套利空间压缩
3. 手续费假设较高
4. 需要进一步参数优化

## 结论与建议

策略逻辑正确实现，但在当前参数下收益有限。建议:
1. **降低手续费假设**（如机构费率万0.5）重新测试
2. **优化参数**: 持仓周期（5→10/20天）、Z-Score lookback（20→60/120天）
3. **品种筛选**: 只保留多空差为正的高流动性品种
4. **加入过滤条件**: 只在高波动率或特定市场状态下交易
