# 恒温器交易策略：原理、应用与Python实现

## 一句话摘要

用 CMI（Choppy Market Index）判断市场处于震荡或趋势状态，自动在"震荡市均值回归"与"趋势市通道突破"两套逻辑间切换，兼顾不同行情的收益并平滑回撤。

## 核心原理

### 1. 市场状态判定（CMI指标）

CMI 用于区分趋势与震荡，核心是"价格净变动 / 期间总波动"，量化趋势强度：

- **公式**: CMI = 100 × |收盘价 - N日前收盘价| / 期间价格总波动（累计TR），常用 N=20，阈值=20
- **判定规则**:
  - CMI < 20：震荡市，价格无明确方向，适合均值回归
  - CMI ≥ 20：趋势市，价格有惯性，适合趋势跟踪
- **关键价**: (最高价 + 最低价 + 收盘价) / 3，用于震荡市的模式识别

### 2. 双模式核心逻辑

| 市场状态 | 核心思路 | 核心工具 | 收益来源 |
|---------|---------|---------|---------|
| 震荡市 | 高抛低吸，区间交易 | 关键价、ATR | 价格围绕中枢来回波动 |
| 趋势市 | 顺势追涨杀跌 | 布林通道、EMA | 趋势延续的惯性收益 |

## 详细交易规则

### 震荡市（CMI < 20）规则

1. **计算关键价**: 昨日关键价 = (昨日High + 昨日Low + 昨日Close) / 3
2. **模式识别**:
   - 今日收盘价 > 昨日关键价 → 趋卖市，优先逢高做空或平多
   - 今日收盘价 ≤ 昨日关键价 → 趋买市，优先逢低做多或平空
3. **开仓触发**:
   - 趋买市：开盘价上方 N倍ATR（N=0.5）挂多单，突破成交
   - 趋卖市：开盘价下方 N倍ATR（N=0.5）挂空单，突破成交
4. **风控**: 止盈 M倍ATR（M=1.5），止损用反向关键价或1倍ATR

### 趋势市（CMI ≥ 20）规则

1. **通道选择**: 布林通道（20日SMA中轨，2倍ATR上下轨）或肯特纳通道
2. **开仓触发**:
   - 多单：价格突破上轨且中轨向上
   - 空单：价格跌破下轨且中轨向下
3. **持仓转换**: 震荡市持仓在趋势确认后，按趋势方向处理
4. **风控**:
   - 止损：多单=开仓价 - 1×ATR，空单=开仓价 + 1×ATR
   - 止盈：固定2×ATR（风险收益比1:2）或跟踪止盈（跌破中轨平仓）

### 状态切换与冲突处理

- **切换触发**: CMI连续2日突破/跌破20，确认状态切换
- **持仓冲突**: 震荡市持仓在趋势市时，按趋势方向清仓或转趋势持仓，避免反向暴露
