## 一句话总结

基于学术论文的期货期限结构差价反转策略：近月（F1）与次近月（F2）合约的周度价差收益率存在显著负自相关，高差价后一周倾向反转回归，构建市场中性的多/空主/超低组合。

## 核心策略逻辑

### 理论基础

- 论文假设：期货曲线各合约对新闻反应速度不同
- 近月（F1）通常反应更快，容易**过度反应**
- 次近月（F2）反应滞后
- 两者之间产生**短暂的错配**，可以被系统性交易

### 策略规则（非常明确）

| 步骤 | 规则 |
|------|------|
| **频率** | 每周五 |
| **计算** | 计算各品种 F1 - F2 的周度价差收益率 |
| **排名** | 按价差收益率从小到大排序 |
| **做多** | 价差最低（最负）的 **4 个品种** → 预期反弹 |
| **做空** | 价差最高（最正）的 **4 个品种** → 预期反转 |
| **权重** | 等权重 |
| **持仓** | 持有一周 |
| **组合** | 每周5个多头 + 5个空头 = 市场中性 |

### 实证结果

| 组合 | 描述 | 年化收益 |
|------|------|---------|
| Portfolio 1 (最低) | 价差最负 | 趋于显著正收益 |
| Portfolio 5 (最高) | 价差最正 | 趋于显著负收益 |
| Low - High 差额 | 反转套利 | ~10% 年化，t-stat = 3.91 |

**回测统计 (2007–2025)**：
- 年化收益: **9.68%**
- 夏普比: **0.92**
- 波动率: **10.6%**
- 最大回撤: **-17%**
- Beta: **≈ 0** (市场中性)
- 收益偏度: **正偏** (更多上行离群值)

## 为什么是有效的异常

1. **单调性模式 (Monotonic Pattern)**：从Q1到Q5不是随机分散，而是平滑递减 → 真实稳定异常，非偶然
2. **负自相关稳健**：高价差预测低价差，低价差预测高价差
3. **跨市场**：股指期货 + 商品期货（金属、能源、农业、畜牧）均有效
4. **无需预测**：不需要基本面、机器学习或前瞻性预测

## 复现可行性评估

| 维度 | 评估 |
|------|------|
| **策略类型** | 期货期限结构差价/跨期套利（非单向CTA） |
| **数据需求** | ⚠️ 需要期货合约的期限结构数据（F1 + F2价格） |
| **参数明确度** | ✅ 非常明确（每周4对4等权） |
| **框架适配** | ⚠️ cta-strategy-factory 为单品种单向框架，无多合约期限结构数据 |
| **近似方案** | 用现货 vs 远期ETF/ETN的basis替代；或用单品种价差代替多品种套利 |
| **执行考量** | 需要同时交易4组多/空（共16合约），周转换 |

## 延伸思考

1. **周频过高**？ 是否可降为月频或双周频以降低滑点和交易成本
2. **动态权重**？ 是否可用价差幅度加权（越极端价差 → 越大仓位）
3. **Regime Filter**？ 高波动率时期价差反转是否更强
4. **跨资产类别**：论文原本使用商品，但QuantReturns扩展到股指也有效

### 差距分析

1. **数据时间范围**：论文从 2015 开始，包含 2017 牛市；我们从 2019 开始，错过了最大一段涨幅
2. **交易成本**：论文 Table 1 未扣成本；我们包含 5bps 手续费 + 滑点
3. **框架限制**：论文有做空扩展（Appendix），我们只做多
4. **ETH 回撤**：-71% 说明 vol targeting 在极端波动市场滞后性严重，不适合单品种大仓位
5. **GLD 是最佳近似**：低波动率资产下 ensemble + vol targeting 效果稳定
