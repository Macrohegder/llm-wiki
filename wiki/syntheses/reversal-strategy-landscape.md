## 二、反转策略类型学

我们将已复现的反转策略按机制分为 4 类：

### 类型 1：价格结构型反转 — IBS/LLV 类
**核心逻辑：** 利用日线价格结构（高低位置 + 相对极值）判断超卖

| 策略 | 信号机制 | 持有期 | 信号频率 | 代表性的最佳品种 |
|------|---------|-------|---------|----------------|
| **FDLR** (5-Day Low Range) | IBS<0.25 + close<LLV5 | 5天 | ★★★ (20/yr) | XLP, EEM, XLK |
| **FiveDayLow** | close=5日最低 | 隔夜 | ★★★ (40/yr) | 防守板块 |
| **IBS Mean Reversion** | IBS<0.1/0.2 | 1-5天 | ★★ (15/yr) | 防守板块 |
| **Consecutive Down Days** | 连跌N天 | 1-5天 | ★★★ (25/yr) | 宽基指数 |

**品种适配分析：**
- **防守板块 (XLP, XLU, XLV)** → ★★★★★：低波动 + 弱趋势使价格结构信号更可靠
- **新兴市场 (EEM, VWO)** → ★★★★：价格结构信号在非效率市场更有效
- **宽基指数 (SPY, QQQ)** → ★★★：受长趋势干扰，信号质量中等
- **高波动商品 (USO, UNG, SLV)** → ★：趋势跳跃破坏结构信号，IBS 无意义

### 类型 2：RSI 超买超卖型
**核心逻辑：** 短期 RSI 超买/超卖 + 趋势过滤

| 策略 | 信号机制 | 趋势过滤 | 持有期 | 代表性的最佳品种 |
|------|---------|---------|-------|----------------|
| **PB** (Pullback MR) | close<SMA20 + RSI5<45 | close>SMA200 | 灵活 | GLD(1.06), XLK(0.72) |
| **RSI2 MR** | RSI2<5/10 | close>SMA200 | 2-5天 | Nasdaq-100 (多标的) |
| **Williams %R MR** | Williams%R < -80 | close>SMA200 | 灵活 | 宽基指数 |
| **RSI Mean Reversion** | RSI<30/35 | close>SMA200 | 灵活 | 宽基指数 |

**品种适配分析：**
- **黄金 (GLD)** → ★★★★★：均值回归最强，PB 策略 Sharpe 1.06
- **科技板块 (XLK)** → ★★★★：高波动反转空间大
- **工业 (XLI, XLY)** → ★★★★：周期板块回调幅度可控
- **宽基指数 (SPY, QQQ)** → ★★★：有效但信号偏少（趋势过滤严苛）
- **防守板块 (XLP, XLU)** → ★★：低波动导致 RSI 信号过少

### 类型 3：趋势+回调型 — SMA200 过滤类
**核心逻辑：** 只在上升趋势中做回调买入，下降趋势中不做

| 策略 | 回调信号 | 出场 | 代表性的最佳品种 |
|------|---------|------|----------------|
| **PB** (Pullback MR) | close<SMA20 + RSI5<45 | RSI5>65 | GLD, XLK, XLI, IWM |
| **RSI2 MR** | RSI2<5/10 | close > 前日高 | QQQ (多标的) |
| **Triple RSI** | 3 个 RSI 周期同时超卖 | RSI 密集 | 宽基指数 |

**规律总结：**
- SMA200 过滤确保了「只买牛市回调」，跳过了 2008/2022 等熊市，这是所有 SMA200 过滤策略回撤小的根本原因
- 但代价是信号稀缺——在长牛市中，close>SMA200 几乎天天成立，但 close<SMA20 的「回调」出现频率远低于预期
- 对防守板块（XLP, XLU, XLV），SMA200 过滤几乎等于永远不过滤（这些板块很少跌破 200MA），所以信号质量取决于回调信号本身

### 类型 4：缺口/结构反转型 — 日历/形态类
**核心逻辑：** 利用特定 K 线形态或日历效应

| 策略 | 信号机制 | 状态 |
|------|---------|------|
| Unfilled Gap | 当日全在昨日区间外 | 信号极少 (25次/30yr) |
| Reversal Day | 大跌后长阳反转 | 付费墙，未复现 |
| Four Up Days | 4连涨后做空 | 付费墙，未复现 |
| 123 Pattern | 三浪反转形态 | 付费墙，未复现 |

**这类策略信号太少，不适合 batch 挖掘。**

## 三、品种-策略匹配矩阵

### 最佳匹配：防守板块 × 价格结构型

```
XLP + FDLR = Sharpe 0.82
XLU + FDLR = Sharpe 0.37 (信号不足)
XLV + FDLR = Sharpe 0.33
```

防守板块（消费必需品、医疗、公用事业）低波动 + 弱趋势的特性，使得价格结构信号（IBS + LLV）极为可靠。特别是 XLP——消费者必需品在回调到 5 日低点时几乎必然反弹。

### 最佳匹配：黄金 × RSI 回调型

```
GLD + PB = Sharpe 1.06
GLD + FDLR = Sharpe 0.30
```

黄金的均值回归特性在所有品种中最强。RSI 回调型策略（PB）在黄金上取得了我们见过的最高 Sharpe 之一（1.06）。但价格结构型（FDLR）效果一般——因为黄金的日线高低点跳跃性大，IBS 信号不可靠。

### 最佳匹配：新兴市场 × 价格结构型

```
EEM + FDLR = Sharpe 0.74
EEM + PB   = Sharpe 0.57
```

新兴市场的非效率性使得价格结构信号（IBS + LLV）效果突出。FDLR 在 EEM 上 Sharpe 0.74，显著高于宽基指数。

### 最佳匹配：宽基指数 × 趋势+回调型

```
SPY + PB    = Sharpe 0.62
QQQ + PB    = Sharpe 0.55
IWM + PB    = Sharpe 0.62
```

宽基指数的长期上涨趋势，使得 SMA200 过滤的回调买入策略（PB）稳定有效。适合作为组合核心组件。

### 最差匹配：商品期货 × 任何反转策略

```
USO + FDLR = Sharpe -0.41
USO + PB   = Sharpe 0.34 (勉强)
XLE + FDLR = Sharpe -0.22
UNG + FDLR = Sharpe 0.16
```

商品（原油、天然气）和能源板块的趋势跳跃性极强，均值回归信号被连续的趋势运动破坏。用反转策略做商品波段是反模式的。

## 四、关键洞察与设计原则

### 原则 1：波动率水平决定策略类型

| 品种波动率 | 适合的反转策略 | 解释 |
|-----------|--------------|------|
| 低 (XLP, XLU, SHY) | 价格结构型 (FDLR, IBS) | 低波动下 IBS/LLV 信号干净 |
| 中 (SPY, QQQ) | 趋势+回调型 (PB, RSI2) | SMA200 过滤去除假信号 |
| 高 (GLD, XLE) | RSI 超买超卖型 | 高波动提供足够的反转空间 |
| 极高 (USO, UNG) | ❌ 不适合反转 | 趋势性 > 均值回归性 |

### 原则 2：趋势强度 × 反转策略的交互

```
趋势弱 + 波动低 = 价格结构型最佳（防守板块）
趋势强 + 波动中 = 趋势过滤型最佳（宽基指数）
趋势弱 + 波动高 = RSI 超买超卖型最佳（黄金）
趋势强 + 波动高 = 不适合反转（能源、商品）
```

### 原则 3：所有反转策略的共同局限

1. **绝对收益低** — 所有已跑 batch 的反转策略年化收益都在 0.01%-0.3% 之间，无法独立作为收益来源
2. **回撤极低** — Sharpe 虽高，但这是低波动带来的假象，不是高超额收益
3. **牛市表现差** — 反转策略在大牛市中表现最差（2021 年 QQQ 上涨 27%，反转策略基本空仓）
4. **熊市是价值期** — 2008、2022 是反转策略最赚钱的时期，波动率飙升提供大量反转机会
5. **最佳用途** — 作为 **风险对冲/尾部保护** 组件，而非收益驱动型策略

### 原则 4：品种选择比参数优化更重要

在同一反转策略下，品种间的 Sharpe 差异远大于参数优化带来的提升：

```
PB 策略：GLD(1.06) vs XLRE(0.15) = 7 倍差异
FDLR 策略：XLP(0.82) vs USO(-0.41) = 无限差异
```

**结论：** 反转策略不挑参数，挑品种。选对品种比参数优化重要 10 倍。

## 五、CTA 组合建议

### 反转策略模块（建议占组合 20-30%）

| 组件 | 策略 | 品种 | 权重 | 预期 Sharpe | 角色 |
|------|------|------|------|------------|------|
| 核心反转 | PB | GLD | 5% | 1.06+ | 最佳单品种反转 |
| 核心反转 | PB | SPY, QQQ, IWM | 10% | 0.55-0.62 | 宽基回调买入 |
| 防守反转 | FDLR | XLP, EEM | 5% | 0.74-0.82 | 低波动保护 |
| 周期反转 | PB | XLK, XLI, XLY | 5% | 0.50-0.72 | 板块回调 |
| 现金储备 | — | — | 5% | — | 灵活性 |

### 红线规则
- 不在能源/商品上使用任何反转策略（XLE, USO, UNG）
- 不在极致牛市中单独依赖反转策略（2020-2021 风格）
- 反转策略不应超过总组合的 30%（收益天花板过低）
