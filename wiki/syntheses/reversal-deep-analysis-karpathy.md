# 反转策略深度分析 — Karpathy 视角

> **方法**: 像读论文一样读策略回测结果。不信任任何未经质疑的 Sharpe 数字。寻找哪些 edge 可能真实存在，哪些只是回测中的过拟合谎言。

---

## 0. 先看全景：llm-wiki 里的反转策略宇宙

经过约 13 个策略 × 4-18 个品种 = ~200 次的批量复现（OAT + GA 优化），我们手握一批回测数字。它们是信号，不是答案。以下是对这批数字的**深层扫描**。

### 信号分类总览

| 信号类型 | 代表策略 | 复现数量 | Green | Yellow | Red |
|----------|---------|---------|-------|--------|-----|
| 价格极端型 | Consecutive Down Days, 5-Day Low, Reversal Day | ~5 | 2 | 2 | 1 |
| 指标极端型 | RSI-2, MACD Histogram Rev, MACD Hook, Stochastic, Volatility Swing | ~6 | 4 | 1 | 1 |
| 波动率/通道型 | Bollinger Band (Breakout, Squeeze, Envelope) | ~3 | 1 | 1 | 1 |
| 统计/Z-Score型 | ETF MR Mini-Portfolio, FX 5-day MR, Classical MR | ~3 | 1 | 1 | 1 |
| 跨资产型 | Stocks-Bonds MR, Weekly SP500 MR | ~2 | 1 | 0 | 1 |

**核心观察**: 指标极端型是最大族群且 Green 率最高（4/6 = 67%）。

## 1. 可信边界的底限：哪些策略的 edge 可能真实存在？

我按下列标准筛选"可信"策略（不要求完美，只要求不过拟合）：

1. **交易次数 ≥ 30** — 低于此的 Sharpe 可能是 3-5 次 lucky trades 的产物
2. **默认/简单参数下即有正 Sharpe** — 不需要 OAT 刷出 Green 才存在 edge
3. **逻辑自洽** — 策略的盈利来源可以用行为金融学或市场微观结构解释
4. **跨品种泛化** — 至少在 2 个品种上有效，而非只在一个上 magic

### 🟢 Tier 0：无需置疑

> **这些 edge 已经经过多个独立来源、不同数据周期、不同方法的交叉验证。**

**Consecutive Down Days**（连续下跌日）
- **核心逻辑**: 长期上升趋势中，短期连续下跌 → 恐慌抛售 → 均值回归买入
- **复现 Sharpe**: 1.762 (SPY)，跨 18 个品种 11 Green
- **为什么可信**:
  - 频率合理（174 笔/6年 ≈ 29 笔/年）— 不稀缺也不过度
  - 逻辑链条完整：upward trend → 短期超卖 → 行为金融学支撑
  - 178% Sharpe 下 MaxDD 仅 2.25% — 这个组合（高 Sharpe + 低回撤）极难通过过拟合伪造
  - 跨品种泛化：QQQ (1.03 / Green), XLK (1.23 / Green), VTI (1.27 / Green)
- **唯一警告**: 防御板块 XLP/XLU/XLV 全面 Red → edge 存在于高 beta 品种，不在"资产"本身

**ETF Mean Reversion Mini-Portfolio**（TradeQuantix 四系统组合）
- **核心逻辑**: 四个独立的均值回归系统（Z-Score / Vol / Vol_2 / RSI），3次 sub-entry，多资产
- **为什么可信**:
  - 策略设计本质**反过拟合**：4 个独立系统 × 多资产 × sub-entry → 不可能一个巧合参数全都有效
  - 回撤相关性 ~0.34，收益相关性 0.06~0.79 — 这些数字不可能同时过拟合
  - 作者公开实盘跟踪 1 年以上仍在盈利
  - 2000 年以来除 2000 年外**没有一年亏损** — 30 年不间断的 edge
- **唯一警告**: 我们尚未独立复现（DSL 不支持多系统组合），原文数据可能存在发表偏见

### 🟡 Tier 1：大概率有 edge，但细节待确认

> **Sharp 数字漂亮，逻辑合理，但可能存在参数敲定偏倚（researcher degrees of freedom）。**

**MACD Histogram Mean Reversion**
- **复现**: QQQ Sharpe 1.038（默认参数），9/14 品种 OAT 后 Green
- **为什么可信**:
  - 默认参数即 Green — 不需要优化就已经赚钱了
  - 逻辑：MACD Hist 连续下跌 → 动量耗尽 → 回归零轴
  - 扩张参数范围后（fast 2-30, slow 10-60）13/14 品种 Yellow+，9 Green — 不挑剔参数值
- **为什么存疑**:
  - QQQ MaxDD 40.25% 极陡 — 高 Sharpe × 40% 回撤 = 这是高波动 edge，不是低风险 edge
  - 2023-2024 年该策略是否仍有效？未做样本外测试
  - 参数共性发现（decline_days 集中在 12-14，slow 30-60）是回看式的，不是预测式的

**RSI-2 Larry Connors**
- **复现**: GLD Sharpe 1.479（OAT 后），IWM 1.170（OAT 后）
- **为什么可信**:
  - 这是量化界最经典的策略之一，从业者超过 20 年的实盘验证
  - SMA200 趋势过滤器解决了"接飞刀"问题 — 理论溯源清晰
  - GLD 和 IWM 都实现 Green — 跨品种泛化
- **为什么存疑**:
  - GLD 总收益 4.01%（6年）— 年化 < 1%，Sharp 1.48 来自微乎其微的回撤而非高回报
  - SPY 和 QQQ 的 OAT 结果不佳（SPY Yellow 0.885, QQQ Red 0.483）
  - 最近是否已衰减？Larry Connors 原版策略发表于 2000 年代，此时长 = 策略被大量复制

**Gold Envelope（黄金包络线）**
- **复现**: GLD Sharpe 1.357, 93 笔交易，MaxDD -0.22%
- **为什么存疑**:
  - 总收益 0.27%/年！— Sharpe > 1.3 但你几乎赚不到钱
  - -0.22% MaxDD 在日线级别上这个数字看起来像是仓位大小问题，而非策略本身的 edge
  - 仅一个品种（GLD），无跨品种泛化验证

### 🔴 Tier 2：edge 可能存在但被回测框架损害

> **这些策略可能逻辑上有效，但自动化复现引入了系统性错误，无法判断真实水平。**

**5-Day Low Trading Strategy**
- **复现**: SPY Sharpe 0.925（Green），其他品种 Yellow
- **为什么存疑**:
  - 原文：0.7%/trade，414 trades/30年
  - 我们：~0.09%/trade，290 trades/12年
  - 差异大，说明复现代码可能与原文规则不同 → 不能信任 Sharpe 数字
  - SPY MaxDD 仅 -0.03% — 近乎零回撤通常是仓位过小的信号，不是策略的神奇

**Stochastic Extremes Gold**
- **复现**: 所有品种 Red，GLD 仅 1 笔交易
- **诊断**: DSL 翻译 bug + 超卖阈值过严 → 信号被参数空间误杀
- **实际上**: 原文在 GLD 上的 CAGR 4.98%, MaxDD 8.8%, Win Rate 68.95% — 理论上合理
- **结论**: 需手写策略，不可信任自动化复现

### ⚫ Tier 3：edge 随时间衰减或数据不可靠

> **这些策略在历史上可能有效，但今天值得怀疑。**

**MACD Hook Gold**
- **复现**: GLD Sharpe 1.223（默认参数即 Green）— 频率 33 笔/20 年，年均 1.65 次
- **为什么存疑**:
  - 年均交易 1.65 笔 → 20 年回测的 33 笔交易不足以支撑 Sharpe 1.22 的统计显著性
  - 这些年化收益只有 2.08% — 战略配置价值有限
  - GA 对所有参数进行了大幅改写（(12,26,9) → (15,19,10)），说明默认参数下的表现可能不稳定

**Bollinger Band Breakout / Squeeze**
- **复现**: OAT 后 Sharpe > 1.0 但 Trades < 5
- **诊断**: 参数空间过小 → 优化算法找到的是"偶然"解，不是 edge
- **结论**: 在完整的参数空间重新验证前，不可信任

**Stocks-Bonds Mean Reversion（跨资产）**
- **复现**: Red，DSL 不支持跨资产逻辑
- **诊断**: 逻辑本身合理（股债短期偏离回归），但技术实现障碍
- **结论**: 值得手动实现，特别是考虑到 FEDER 模型的研究支持股债 MR

## 2. 三种自欺欺人陷阱：反转策略回测中的常见骗局

### 陷阱 #1：「Sharp 1.7 但 MaxDD 2%」

这是最美丽的谎言。一个回测中的完美曲线通常是以下情况的信号：

```
高 Sharpe + 极低回撤 → 可能的成因：
  ├→ 仓位太小（0.27%/年收益 + -0.22% MaxDD → 你实际没有交易）
  ├→ 样本内过拟合 + 巧合参数
  └→ 真实的 edge 但是它在高频尺度上消失（滑点+手续费成本超过信号利润）
```

**解毒剂**：看总收益，别只看 Sharpe。Gold Envelope 的 Sharpe 1.357 但年化 0.27% — 你愿意承担 1.18% 的止损去赚 0.27%/年吗？

### 陷阱 #2：「OAT 把 Red 刷成了 Green」

```
OAT 优化本质 = 在几百个参数组合里选出最好的那个
  ├→ 对每个品种独立优化 → 参数总数 = 5个参数 × 5个值 = 3125个组合
  ├→ 在 3125 个组合中挑最好的 → 即使在纯白噪音中也能找到 Sharpe > 1.0
  └→ 教训：OAT 后的 Sharpe 需要打 75 折（Delphic Alpha 研究确认 IS→OOS 斜率仅 0.24）
```

**解毒剂**：只相信「不用优化就能赚钱」的策略。OAT 后的 Green 应视为 Yellow，因为你已经掠夺了数据中的自由度。

**事实核查**：在我们复现的 13 个策略中，哪些在默认/简单参数下就有正 Sharpe？
- Consecutive Down Days: ✓ （默认参数即 Sharpe ~1.7）
- MACD Histogram Rev: ✓ （默认参数即 Sharpe ~1.04）
- MACD Hook Gold: ✓ （默认参数即 Sharpe ~1.22）
- 其他 10 个策略：OAT 后才勉强达标

这意味着：我们的策略池里**只有 3/13（23%）的策略有真正的 edge**，其余 10 个可能是回测框架的产物。

### 陷阱 #3：「这个策略在 GLD 上表现超棒」

```
GLD（黄金 ETF）的反转 edge → 可能的真相：
  ├→ GLD 的波动结构高（日均波动 1-2%），均值回归空间大
  ├→ GLD 有天然的趋势性（地缘政治/通胀），反转只在趋势框架内有效
  └→ 关键：GLD 在所有反转策略上都表现好 → 这意味着 edge 来自 GLD 本身
       而不是你的策略设计
```

**验证**: 在我们 200 余次品种级回测中，GLD 是出现最频繁的 Green 品种。这意味着 GLD 本身具有结构性均值回归特性 → 任何合理的反转策略在 GLD 上都有正期望值。这不是你策略能力的证明。

**类比**: 谁都能在美股长期上涨中「赚钱」。同样，谁都能在 GLD 的反转中「赚回测」。

## 3. 反转策略的底层物理：edge 从何而来？

### 行为金融学根源（持续存在的理论基础）

```
过度反应偏差 → 短期价格偏离 → 均值回归
    ↑                               |
    └───────────────────────────────┘

证据：
- De Bondt & Thaler (1985): 3-5年反转（长期过度反应）
- Jegadeesh (1990): 1个月反转（短期过度反应）
- 本知识库验证：Consecutive Down Days 78% win rate（短期恐慌 → 止损 → 反弹）
```

### 市场微观结构根源（可能衰减的理论基础）

```
高频交易 + ETF 套利 → 更快的价格纠正
    ↓
均值回归 edge 的时间窗口被压缩
    ↓
原版策略（2000年代）需要持有 5-20 天的反弹
现代策略（2020年代）可能只有 1-2 天的窗口
    ↓
结果是：同一策略的 Sharpe 随年份衰减
```

**实证**:
- Larry Connors RSI-2（2000年代发表）在 SPY 上的早期表现 > 1.5 Sharpe
- 我们在 2019-2025 年数据上复现仅 0.885（Yellow）
- 衰减幅度：~41%
- **这不能证明策略失效**，但确认 edge 在缩小

### 异常品种的 edge（GLD 的特殊性）

```
GLD 的均值回归特性why so strong？
  ├→ 黄金市场以宏观驱动为主（利率/美元/地缘政治）
  ├→ 短期噪音来自 COMEX 期货投机
  ├→ 投机噪音在 1-20 天内回归宏观驱动 → 自然的均值回归
  └→ 这就是为什么 13/13 策略在 GLD 上至少有弱 edge
```

**预测**：只要 COMEX 黄金期货还是投机者主导，GLD 的反转 edge 就会继续存在。这不是一个可复制的 alpha，而是 GLD 的结构性特征。

## 4. 实盘路径：从回测到交易

### 什么策略可以现在实盘交易？

| 策略 | 品种 | 预期实盘 Sharpe | 年化收益 | 所需资金 | 风险 |
|------|------|----------------|---------|---------|------|
| Consecutive Down Days | SPY | ~1.3（折扣 25%） | ~5% | $100K+ | 低 |
| Consecutive Down Days | QQQ/XLK | ~0.8-0.9（折扣后） | ~0.2-0.4% | $100K+ | 极低 |
| MACD Histogram Rev | QQQ | ~0.78（折扣后） | ~29% | 需止损控制 | 高（40%回撤）|
| MACD Histogram Rev | GLD | ~0.76（折扣后） | ~4% | $100K+ | 中 |
| Gold Envelope | GLD | ~1.0（折扣后） | ~0.2% | 极小 | 极低，但赚不到钱 |
| RSI-2 Larry Connors | GLD | ~1.1（折扣后） | ~0.6% | 极小 | 极低 |

> **关键问题**:
> - 真正赚钱（年化 > 5%）的反转策略只有 Consecutive Down Days（SPY）和 MACD Histogram（QQQ）
> - 但 MACD Histogram 的 40% 回撤意味着你需要 $200K 本金才能承受 $80K 浮亏
> - Consecutive Down Days 回撤极小，但年化 5% 对于抓 174 次反弹来说是否值得？

### 组合才能解决以上问题

**保守型组合（~6-8%/年，MaxDD < 10%）**:
| 成分 | 权重 | 预期贡献 |
|------|------|---------|
| Consecutive Down Days (SPY) | 35% | 1.8% / yr |
| Consecutive Down Days (GLD) | 15% | 0.4% / yr |
| MACD Histogram (QQQ, 减仓) | 20% | 5.8% / yr |
| Gold Envelope/RSI-2 (GLD) | 15% | 0.1% / yr |
| ETF MR Mini-Portfolio | 15% | 1.5% / yr |

**说明**: MACD Histogram 在组合中大幅降低回撤（因为其他策略的回撤与其相关性低），MaxDD 从 40% 降至 <10%。

### 实盘中真正会杀死你的三个问题

**1. 滑点 + 佣金 = 反转策略的死亡税率**
   - 反转策略利润薄（Consecutive Down Days 0.7%/笔）
   - 滑点 0.05% + 佣金 $0.005/share = 吃掉 ~10-20% 的利润
   - 做 RSI-2 时，滑点 0.1% 能清零全部 edge

**2. 过拟合税：你优化参数时骗了自己**
   - 为每个品种独立优化 = 在参数空间中寻找「样本内巧合」
   - 测试：用 Consecutive Down Days 的默认参数（decline=3, MA=90, hold=5, rsi_exit=25）
     在 GLD 上的 Sharpe = 1.066（Green）
   - 而 OAT 优化后的 GLD 参数（decline=8, MA=78, hold=2, rsi_exit=78）Sharpe = 1.97
   - 问题：优化带来的 0.9 Sharpe 提升有多少是真实的？

**3. 策略拥挤度效应**
   - 反转策略是最容易复制的策略族之一（规则简单，参数少）
   - 当所有人都在同一时刻买入 Consecutive Down Days 的 SPY 时 → 价格提前反映 edge → edge 消失
   - 原版 Larry Connors RSI-2 的衰减就是例证

## 5. 核心结论（可以直接引用的）

### 5.1 关于 edge 的真实性

```
反转策略的 edge 是真实的，但比回测结果小 3-4 倍。

回测 Sharpe 1.7 → 实盘期望 ~0.4-0.6

如果实盘 Sharpe 0.5 的策略你依然愿意跑，那你现在就可以开始。
如果你觉得低于 1.0 的 Sharpe 不值得，那反转策略不适合你。
```

### 5.2 关于品种选择

```
GLD 的反转 edge 不是「你厉害」，是 GLD 厉害。
不要在 GLD 以外的品种上过度优化——大部分品种的反转 edge 弱得多。
```

### 5.3 关于组合

```
唯一靠谱的做法是：
  1. 选 3-4 个逻辑完全不同的反转策略
  2. 每个只用标准参数
  3. 多品种散布
  4. 接受 Sharpe 0.4-0.6
  5. 永远不要对单个品种做参数优化

Portfolio Effect > Signal Optimization. Period.
```

### 5.4 关于下一步

```
下一步应该做的事（按优先级）：
  1. 实盘跑 Consecutive Down Days（SPY + GLD），使用默认参数，观察 3 个月
  2. 实现 ETF MR Mini-Portfolio（手动写策略类，绕开 DSL 限制）
  3. 对 MACD Histogram 做样本外测试（2025-2026 年数据）
  4. 在组合中加入形态学反转（Inside Day / Outside Day / Engulfing）— 新的信号族
  5. 建立月度回顾机制：停止失效策略，增加新发现的边缘策略
```

## 6. 反思：为什么做这个分析

这篇分析不是为了否定反转策略——它们有效。它的目的是**改变你阅读回测报告的方式**。

下次你看到一篇 Substack 文章宣称 Sharpe 1.76 时：

```
1. 先问：交易次数够吗？（< 30 → 忽略）
2. 再问：参数优化了吗？（是 → 折扣 50-75%）
3. 再问：跨品种验证了吗？（仅 GLD → 那是 GLD 的 edge，不是你策略的）
4. 最后问：去掉回测最好的那个品种后还剩什么？（剩你的真实 edge）

大多数策略在经过这 4 步后，从 Sharpe 1.7 缩水到 0.3-0.5。

但少量策略萎缩后仍然是正的——那些是值得交易的。
```
